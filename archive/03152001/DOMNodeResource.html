DOMNodeResource Module

DOMNodeResource provides DOM API access to an underlying resource.

DOMNodeResource

The interface defines no new methods and simply overrides those in Resource. The type of the stored content is a DOM Node so the return value and arguments of getContent and setContent must be cast to the appropriate language specific DOM Node representation.
